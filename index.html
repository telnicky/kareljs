Welcome, I am Karl the robot. My world contains green beepers and I need your help to organize them. On the screen you have a text editor, the code you write in the text editor will help me navigate through my world.

Remember karl understands javascript, so don't forget to use if statements, while loops, or for loops!

  • run: karl will load your code and run it.
  • reset: once a level is completed it will not reset, use this button to reset the level.
  • solution: display where karl wants you to place the beepers
  • help: toggle this menu.
  • level-select: change levels.
